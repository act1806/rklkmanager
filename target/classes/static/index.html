北京荣科利康科技有限公司

北京荣科利康科技有限公司创办于2013年，与英国BioVet-Lab公司合作，引进其动物生物制品的关键技术及相关产品。公司是集科研、开发、生产、服务于一体的综合性高新生物技术和生物制品研发企业。 公司一贯秉持“从源头着手，预防动物感染人；保障人与动物的幸福相处”的理念，致力于人与动物和谐健康相处；我们提倡“早检测、早发现、早治疗”，为您和爱宠提供健康保障。


目前，公司主要致力于宠物疾病诊断产品和服务领域，包括动物疾病快速诊断检测卡，第三方精准检测技术服务，qPCR快速精准检测设备及试剂盒等。同时，公司也在不断引进和开发动物治疗药物，动物健康产品等。 公司聘请了中国农业大学和华中农业大学等多位著明兽医师和动物营养专家担任常年的技术顾问，为产品在中国区的推广提供更专业的技术支持。


公司将继续以不断突破创新的精神，不断探索新时期、新环境下的发展战略，凝聚社会助力，服务人类及宠物的健康事业。

公司介绍

北京荣科利康科技有限公司创办于2013年，与英国BioVet-Lab公司合作，引进其动物生物制品的关键技术及相关产品。公司是集科研、开发、生产、服务于一体的综合性高新生物技术和生物制品研发企业。


公司拥有运营、销售、科研等团队。目前共有博士学历7人、硕士学历18人，多人拥有海外留学经历及海内外大型生物医药公司工作经验，团队在2013年进军宠物健康行业，人员队伍逐渐壮大，新生力量不断增多，为企业注入了年轻的活力，在宠物疾病快速诊断、精准检测及快速精准检测方面取得了良好的进展。
